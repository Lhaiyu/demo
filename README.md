#MarkDown概述
##MarkDown定义
- MarkDown是一种轻量级的标记语言
- 专注码字，用[标记]语法，代替常见的排版方法，使普通文本内容具有一定的格式
##MarkDown优点
1. 纯文本内容，兼容所有的文本编辑器与字处理软件
2. 可以放到版本管理系统中，追踪历史变更
3. 轻松的导出HTML、PDF和本身的.md文件
4. 简洁、高效、可读、直观、学习成本低
5. 专注文字内容而不是版本样式，安心写作
##MarkDown用途
1. IT人士：写日志、技术文稿、记录代码片段、撰写文档
2. 科研人员/学生：撰写科技论文、记笔记
3. 新职者：制作求职简历
4. 其他；：撰写博文、在线编辑文章等
##MarkDown实例
- 百度脑图
- 有道云笔记

2. ## 二级标题  
3. **加粗**
